# Bitasmbl-SaaS-Subscription-Tracker-3b99c3-9b15c1dd694d

## Description
Build a small multi-tenant SaaS subscription tracker using ASP.NET Core and React to evaluate junior full‑stack end‑to‑end problem solving.

## Tech Stack
- ASP.NET Core
- React

## Requirements
- Implement secure multi-tenant CRUD for subscriptions with a React dashboard and basic metrics.

## Installation
- git clone https://github.com/MrBitasmblTester/Bitasmbl-SaaS-Subscription-Tracker-3b99c3-9b15c1dd694d.git
- Backend: restore and build ASP.NET Core project
- Frontend: cd into React app and run npm install

## Usage
- Backend: run ASP.NET Core application
- Frontend: run React development server

## Implementation Steps
1. Scaffold ASP.NET Core project and configure multi-tenant subscription model.
2. Implement secure CRUD endpoints for subscriptions.
3. Create React app with authentication-aware dashboard.
4. Integrate CRUD operations from React to ASP.NET Core.
5. Add basic subscription metrics to the dashboard.

## API Endpoints
- /subscriptions (CRUD, multi-tenant aware)
